### MVVM Architecture
- View -> UI Design
- ViewModel -> Logic
- Domain -> Network
- Data Repository -> API Call
- Utils -> Same component push
- Respository -> Backend Data Network
- Resources(res) -> Const value,Difference value etc
- Data -> database
- Model -> API
- lib
   |
   - data
   - res
   - respository
   - utils
   - view
   - view_model
